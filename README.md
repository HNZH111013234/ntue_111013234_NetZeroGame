# 淨零排放宣導小遊戲
<br>
這是一個使用Processing製作的培養淨零排放思維的遊戲，目的是將綠生活資訊與認知以較輕鬆的方式傳遞給民眾，
玩家可點擊與各場景中的物品互動，以及透過小遊戲和使用蒐集的物品與目標配對來完成減碳行動和宣導淨零排放概念和相關知識，並期望使用者能在現實生活中實踐。
備註:需到  tool\library 下載Minim套件才能播放聲音
